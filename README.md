## 주요 작업 및 기능
#### 백엔드
- 제공페이지 : 로그인, 회원가입, 메인페이지, 중고거래 상세페이지
- JWT(access, refresh)를 활용한 회원가입 & 로그인
- 채팅 기능
- 여러 도메인 CRUD
#### 프론트엔드
- 전체 css
- 로그인, 회원가입(세션스토리지)
- 채팅 기능(미구현😭)
- 제품 페이징 처리
- 각 제품당 상세페이지 구현

<br>
